# FinSight Context

## Glossary

### Merchant

A canonical payee identity derived from bank transaction text. Merchant identity applies to all transactions, while merchant spend views filter that identity to expense transactions. Aliases map multiple normalized bank-provided names to one canonical merchant name.

### Recurring Payment

A repeated expense pattern for one Merchant, detected from transaction intervals. Recurring Payment views classify cadence, active status, expected next payment date, and monthly equivalent cost from historical transactions.
